# EPN Workshop Instructions

A Next.js site with setup instructions for the Cursor workshop.

## What participants need

1. [Cursor](https://cursor.com) account
2. [GitHub](https://github.com/signup) account
3. [Vercel](https://vercel.com/signup) account

Workshop template: https://github.com/lmiguelvargasf/cursor_workshop

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

Push this repository to GitHub, then import it in the [Vercel dashboard](https://vercel.com/new). Vercel will detect Next.js automatically.
